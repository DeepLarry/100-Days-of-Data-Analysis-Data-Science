# 🎓 Student Performance Predictor

A simple, interactive web application built with **Python**, **Streamlit**, and **Object-Oriented Programming (OOP)** principles. This app predicts a student's marks based on their study habits and attendance.

## 🚀 Features
- **OOP Structure:** Uses a `StudentPredictor` class to manage data and logic.
- **Real-time Prediction:** Calculates marks instantly based on user input.
- **Performance Feedback:** Categorizes results into *Excellent*, *Good*, or *Needs Improvement*.
- **Modern UI:** Built with Streamlit for a clean, browser-based experience.

## 🧪 The Formula
The prediction logic uses a weighted calculation:
- **Study Hours:** Multiplied by 5
- **Attendance %:** Multiplied by 0.3
- **Assignments:** Multiplied by 2

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Library:** Streamlit
- **Concept:** Object-Oriented Programming (Classes, Methods, Encapsulation)
